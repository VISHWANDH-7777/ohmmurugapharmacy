# Ohm Muruga E-Pharmacy Website

## Overview
Ohm Muruga E-Pharmacy is a web application that allows users to search, view, and order medicines online. It also supports prescription uploads and provides an admin panel for managing inventory and orders.

---

## Features
- User registration and login  
- Search and browse medicines  
- Add to cart and place orders  
- Prescription upload system  
- Order tracking  
- Admin dashboard  

---

## Technology Stack

### Frontend
- HTML  
- CSS  
- JavaScript

- PHP / Node.js
