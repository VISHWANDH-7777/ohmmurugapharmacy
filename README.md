# Ohm Muruga E-Pharmacy Website

## Overview
Ohm Muruga E-Pharmacy is a web application for browsing medicines, managing products, and supporting pharmacy-related workflows.

---

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   npm install
2. Set the GEMINI_API_KEY in .env.local to your Gemini API key if you want Gemini features enabled.
3. Start the app:
   npm run dev

---

## Notes
This repository contains a Vite-based React frontend and a Node/Express backend. The app is configured to run locally on port 3000.
